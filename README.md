# TubeTaker

Um script Python simples para baixar vídeos do YouTube em 720p.

## Requisitos
- Python 3.9+
- yt-dlp
- ffmpeg

## Instalação 

bash

## Como usar
python3 youtube_downloader.py

## Características
- Download de vídeos em 720p
- Salva automaticamente na pasta Downloads
- Gera arquivo MP4 único com áudio e vídeo
